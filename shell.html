About This Project

This dashboard combines a machine-learning model (trained on vegetation, moisture, and slope) with live data (active fires and weather) to provide a real-time situational awareness tool for first responders.

Layers:

  • Predicted Wildfire Risk: The "Fuel" layer (Green to Red) generated by our ML model.
  • Active Fires (24h): The "Spark" layer (Red dots) from NASA's FIRMS satellite.
  • Critical Infrastructure: The "Consequence" layer (Red icons) showing hospitals and schools.
